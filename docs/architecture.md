# Jirow Data Insights Architecture

## MVP Architecture

Frontend:
- Flutter
- Dart

State Management:
- Riverpod

Data Sources:
- Android UsageStatsManager
- Android NetworkStatsManager

Storage:
- Hive or SQLite

Charts:
- FL Chart

Architecture Pattern:
- Clean Architecture
- Repository Pattern

## Future Architecture

Frontend:
- Flutter

Backend:
- AWS Lambda
- API Gateway
- DynamoDB

Authentication:
- AWS Cognito

Analytics:
- Python
- Pandas

Deployment:
- AWS Amplify
